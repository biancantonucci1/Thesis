# Thesis
Code for my thesis

You should run the scripts in this order for maximum efficientcy:

Real_no_SN_cov -> RealSN+CMB+BAO -> w0wa_NOISY -> wNOISY -> w0wa_lessNOISE -> wNOISYlessnoise


Real_no_SN_cov: The script running the real data without supernova covariances

Real_SN+CMB+BAO: The script running the real data with supernova covariances

w0wa_NOISY: Running w0waCDM mock data realisations with full noise

w0wa_lessNOISE: Running w0waCDM mock data realisations with reduced noise

wNOISY: Running wCDM mock data realisations with full noise

wNOISYlessnoise: Running wCDM mock data realisations with reduced noise


Running the Analysis

You can either run all the realisations from scratch or use the precomputed chains.

Option 1 — Run everything from scratch

All realisations can be generated by running the notebooks directly.
Note that generating the full set of realisations may take several hours and may need to run overnight depending on your machine.


Option 2 — Use precomputed realisations

Precomputed .h5 files containing the realisations are available here:
https://liveuclac-my.sharepoint.com/:f:/g/personal/zcqsban_ucl_ac_uk/IgARcevfH9euTZmQ99SsQjCfAVkHVABQPLjMBGona3iRNy4?e=CwtPB4

Download all the files from this folder and place them in the same directory from which you run the notebooks.
If the files are placed in this location, the paths used in the notebooks should work automatically without modification.

You can change the path and names of output files in the sampler code. If you are using the pre-ran h5 realisations, then make sure the path is adjusted to your computer.
